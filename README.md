# Data Analytics Project: Customer Shopping Behavior Analysis

## Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from data ingestion and cleaning to SQL analysis, dashboard development, business reporting, and presentation creation. The objective is to analyze customer shopping behavior, identify trends and patterns, and provide actionable business insights through data-driven decision-making.

---

## Dataset

**Dataset:** Customer Shopping Behavior Dataset

The dataset contains customer demographics, purchasing patterns, product categories, payment methods, subscription details, review ratings, and other shopping-related attributes.

### Key Features

- Customer ID
- Age
- Gender
- Location
- Category
- Item Purchased
- Purchase Amount
- Payment Method
- Review Rating
- Subscription Status
- Discount Applied
- Shipping Type
- Previous Purchases

## Project Workflow

### 1. Data Loading

- Imported the dataset using Python and Pandas.
- Performed initial data inspection and validation.
- Verified data structure and quality.

### 2. Exploratory Data Analysis (EDA)

- Generated descriptive statistics.
- Analyzed distributions of numerical and categorical variables.
- Identified missing values and duplicate records.
- Explored customer demographics and purchasing behavior.
- Created visualizations to identify trends and patterns.

### 3. Data Cleaning

- Handled missing values using appropriate techniques.
- Removed duplicate records.
- Standardized column names and formats.
- Corrected data types.
- Created derived features for analysis.

### 4. SQL Analysis

- Loaded the cleaned dataset into PostgreSQL, MySQL, and SQL Server.
- Performed analytical SQL queries to answer business questions.
- Used:
  - Joins
  - Aggregations
  - CTEs
  - Window Functions
  - Ranking Functions
  - Subqueries

### Sample Analysis

- Top-selling products
- Category-wise sales performance
- Customer purchase trends
- Discount usage analysis
- Review rating insights
- Subscription behavior analysis
- Regional performance analysis

### 5. Power BI Dashboard Development

Created an interactive Power BI dashboard featuring:

- KPI Cards
- Sales Performance Analysis
- Product Category Analysis
- Customer Demographics
- Geographic Insights
- Subscription Analysis
- Discount Impact Analysis
- Review Rating Trends

### 6. Reporting & Presentation

- Prepared a business insights report summarizing key findings.
- Created an executive presentation using Gamma AI.
- Presented recommendations based on analytical findings.
